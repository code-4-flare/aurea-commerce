# Triage Labels

Work items normally use plain lifecycle statuses. If a Matt Pocock engineering skill needs a triage role, use this mapping without creating external labels.

| Skill role | Local status | Meaning |
|---|---|---|
| `needs-triage` | `Needs triage` | Maintainer evaluation is required. |
| `needs-info` | `Needs information` | Work is waiting for missing context. |
| `ready-for-agent` | `Proposed` | Scope is complete enough for implementation. |
| `ready-for-human` | `Needs human` | Human-only judgment or access is required. |
| `wontfix` | `Won't fix` | The work will not be actioned. |
